# 🌍 Универсальный MRZ‑Генератор (ICAO DOC 9303)

Веб‑приложение для генерации MRZ‑кодов паспортов и ID‑карт (TD3 и TD1).  
Поддерживает добавление **Extra Info** — дополнительных данных, которые встраиваются в зону MRZ.
